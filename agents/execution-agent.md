# Execution Agent

## Overview

The Execution Agent is responsible for executing API requests based on the test cases generated by the Test Case Agent.

This agent interacts directly with the API endpoints and records the responses.

---

## Responsibilities

The Execution Agent performs the following tasks:

* Send API requests using generated test cases
* Capture API responses
* Record status codes
* Store response data for further analysis

---

## Process

1. Receive test cases from the Test Case Agent.
2. Execute API requests for each test case.
3. Capture API responses including status codes and response messages.
4. Send the execution results to the Result Analyzer Agent.

---

## Input

Generated test cases from the Test Case Agent.

---

## Output

Execution results including:

* Status code
* Response body
* Error messages (if any)
